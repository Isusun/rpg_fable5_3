# 六輝のオーブ - Six Lights of Eldia - 仕様書

ブラウザだけで遊べる、ドラゴンクエスト風の完全オリジナル2D JRPG。
`index.html` をローカルサーバー経由で開くだけでプレイできます（例: `npx http-server -p 8123` → `http://localhost:8123`）。

- グラフィック: PixiJS v8（CDN）
- 画像アセット: 全8枚を **Codex CLI + gpt-image-2 でローカル生成**（`assets/*.png`、外部URL依存なし）
- BGM/効果音: Web Audio APIによるチップチューン合成（外部音源なし・デフォルトOFF）
- セーブ: localStorage（ブラウザ内で完結）

---

## 1. ゲーム概要（特徴）

- 6つのクラスから主人公を作成し、酒場で仲間を登録・編成する **キャラクターメイク型パーティRPG**
- ワールドマップ＋町5つ＋ダンジョン6つ（洞窟・遺跡・祠・尖塔・火山・魔王城）＋裏世界＋裏ダンジョン
- **6つのオーブ**を集めて魔王に挑む王道シナリオ。クリア後は**裏世界（黄昏のエルデア）**と裏ボスが開放
- 速度順ターン制バトル。敵の隊列（前列/後列）、属性相性、会心、状態異常（毒・眠り・マヒ）
- **属性パズルボス**「氷晶竜フリムド」: まとう衣（炎/氷/雷）の弱点を突くとひるむ
- 押し岩×スイッチのギミック（風唱の祠）、施錠扉×2種のカギ、渡し船、オーブ数で消える関所と結界
- PCキーボード / スマホタッチ（仮想パッド＋メニュー直接タップ）両対応
- `prefers-reduced-motion` 尊重＋「演出ひかえめ」設定、色に依存しない情報提示（前/後ラベル・状態異常は文字表示）

## 2. ストーリー（導入）

> ――星々の光が消えた夜。
> 魔王シェイドヴェインは世界を照らす「六輝のオーブ」を奪い、エルデアは終わらない黄昏に沈み始めた。
> 奪われたオーブは 大地・潮・氷華・焔・風唱 の5つ。最後の「光燿のオーブ」はハーベル王家が守っている。
> 王命を受けた若き冒険者は、酒場で仲間を集め、5つのオーブを取り戻す旅に出る。
> 六輝が揃うとき、南の海の結界は消え、暗黒島の魔王城への道が開かれる――。

クリア後、ハーベル北西の古い祠から「黄昏のエルデア（裏世界）」へ渡れるようになり、すべての元凶・混沌神アズラウスとの真の最終決戦が待つ。

## 3. 操作方法

### キーボード
| キー | 機能 |
|---|---|
| 矢印キー / WASD | 移動・カーソル移動 |
| Enter / Space / Z | 決定・調べる・話す |
| Esc / X / Backspace | キャンセル・**メニューを開く**（フィールド時） |

### タッチ（スマホ）
- 画面左下: 十字キー（移動・カーソル）
- 画面右下: **A**（決定）**B**（キャンセル）
- 画面右上: **メニュー** ボタン
- メニュー項目・敵ターゲット・メッセージ送りは**直接タップ**でも操作可能

### メニュー（フィールドでEsc/メニューボタン）
つよさ / スキル / どうぐ / そうび / ならびかえ / せってい / セーブ / とじる

- セーブはメニューからいつでも可能。タイトルの「つづきから」で再開
- せってい: BGM・効果音のON/OFF（**デフォルトOFF**）、音量10段階、演出ひかえめ。自動保存され再読込後も維持

## 4. システム

### キャラクター・成長
- クラス6種: **ゆうしゃ / せんし / まどうし / そうりょ / とうぞく / ぶとうか**
- ステータス7種: HP / MP / ちから / まもり / すばやさ / かしこさ / うんのよさ
- レベル最大40。レベルアップでステータス上昇・クラス固有スキルを段階的に習得
- 酒場（ハーベル）で仲間を作成（名前かな入力＋クラス選択、最大8人登録）、パーティ入替（最大4人）、メニューで隊列変更
- 隊列: 1〜2番目=**前衛**（物理与ダメ増・狙われやすい）、3〜4番目=**後衛**（物理与被ダメ減）

### 装備
- スロット4: 武器 / よろい / 盾 / 装飾品。クラスごとに装備可能武器種が異なる（剣・斧・杖・短剣・爪・槌）
- 属性武器（炎竜の剣など）、属性耐性防具、状態異常無効の装飾品あり
- 装備画面で変更後ステータスをプレビュー表示

### 戦闘
- シンボルなしのランダムエンカウント（街道はエンカウントなし）
- 全員のコマンド入力 → **すばやさ×乱数**で行動順決定（「しっぷう突き」は必ず先制）
- コマンド: たたかう / スキル / ぼうぎょ（被ダメ半減）/ どうぐ / にげる（ボス戦不可）
- 敵は最大5体・前列/後列の隊列あり。敵ごとにHPバー・状態異常・名前を常時表示、手番は▼マーカー表示
- 属性9種（無・炎・氷・雷・風・地・水・光・闇）。弱点2倍/耐性/無効/吸収、「弱点をついた！」表示
- 会心の一撃（うんのよさ依存）、ダメージポップ・ヒットフラッシュ・画面シェイク演出
- 状態異常: **どく**（毎ターン最大HPの約1/12ダメージ）/ **ねむり**（行動不能、時間経過・被弾で覚醒）/ **マヒ**（行動不能、時間経過で回復）。ボスは状態異常無効
- バフ/デバフ: ちから・まもり・すばやさ±2段階
- 全滅時: 最後に泊まった宿で復活（所持金半分、HP半分）。ゲームは継続可能

### パズルボス「氷晶竜フリムド」
- 【炎の衣】【氷の衣】【雷の衣】を順にまとう。現在の衣と**弱点属性が画面に常時表示**される
- 弱点属性で攻撃 → 2.2倍ダメージ＋**ひるみ（1回行動スキップ）**
- 衣と同じ属性 → **吸収（回復させてしまう）**。その他属性は0.7倍

### 町の施設
- **宿屋**: 全回復（テレポの翼・全滅時の復帰地点になる）
- **道具屋/武器屋**: 購入・売却（売値は半額）。購入時に装備可能クラスを表示
- **教会**: 戦闘不能の仲間を蘇生（Lv×12G）
- **酒場**（ハーベルのみ）: 仲間の作成・入替

### 進行ギミック
- 洞窟のカギ（王から入手）→ 囁きの洞窟の扉
- 古代のカギ（遺跡の宝箱）→ 風唱の祠の扉
- 渡し船の手形（潮のオーブ後、船主から）→ 南の砂の大陸への渡し船
- 北の橋の衛兵 → **オーブ3つ**で通してくれる（雪国へ）
- 六輝の結界 → **オーブ6つ**で消滅し魔王城へ
- 風唱の祠: 岩2つを封印の台座に押し込むと結界が開く（岩はマップ入り直しでリセット＝詰み防止）

## 6. クリアまでの道筋（攻略チャート）

1. **ハーベル城下町**: 王に謁見→「洞窟のカギ」入手。酒場で仲間を3人作成（戦士・僧侶・魔導士推奨）。装備を整える（想定Lv1〜4）
2. **囁きの洞窟**（東の山脈）: 最深部の扉をカギで開け、大地の間へ。**ボス: ガンド**（雷が弱点・物理耐性。想定Lv5〜6）→ **大地のオーブ**
3. 東の国へ。**カルスト**で装備更新 → **港町メリア** → 南東の砂浜から**沈んだ遺跡**へ。B1の宝箱で**古代のカギ**入手。**ボス: リヴァル**（雷弱点。眠り対策にめざましベル。想定Lv9〜11）→ **潮のオーブ**
4. メリアの船主から**渡し船の手形**→ 南の砂の大陸へ。**ザーラ**で補給
5. 西大陸へ戻り北の**風唱の祠**（古代のカギで開錠）。岩2つを台座に押し込み結界を解除。**ボス: ゼフ**（地属性が有効・風無効。想定Lv12〜13）→ **風唱のオーブ**
6. 砂漠西の**燃えさかる深淵**。**ボス: イグナス**（氷・水弱点、炎吸収。想定Lv15〜16）→ **焔のオーブ**
7. オーブ3つ以上で北の橋の衛兵が通してくれる → 雪の町**フレル** → **氷晶尖塔**。**ボス: フリムド**（衣の弱点表示を見て属性を切替。氷の衣=炎、炎の衣=氷、雷の衣=地クエイク。想定Lv17〜19）→ **氷華のオーブ**
8. オーブ5つでハーベル王へ → **光燿のオーブ＋光燿の聖剣**拝領。南の海の**結界が消滅**
9. 暗黒島の**魔王城**へ。玉座で**魔王シェイドヴェイン**（第1形態→真の姿の連戦。闇耐性装備・ヒールガ・ときの声/ガードラ活用。想定Lv22〜25）→ **エンディング**（クリアデータ保存）
10. **裏世界**: ハーベル北西の祠から黄昏のエルデアへ。生き残りの商人から最強装備を購入し、南の**虚無の聖域**へ。**裏ボス: アズラウス**（2回行動・自己回復。想定Lv28〜32）→ **真エンディング**

### 攻略のコツ
- ぼうぎょは被ダメ半減。ボスの全体攻撃が続くターンは後衛を守る
- ボスは状態異常無効だが、**デバフ（よろいくだき・スロウダスト）は有効**
- 弱点を突くと1.5〜2倍。「効きがわるい…」が出たら属性を変える
- 全滅してもゴールド半分で宿に戻るだけ。こまめなセーブ＋テレポの翼で探索効率UP
